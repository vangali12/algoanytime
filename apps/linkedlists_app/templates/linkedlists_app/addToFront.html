{% load static %} Add To Front

[Count Sort]

Input: Value of new node

Output: The Sll instance

Just like the "Add Node" function, except this algorithm will add the node to the front of the Sll.

Make sure the return value is the instance of the Sll to allow for chaining of node additions.

Edge Cases:

What do you do when you are given an empty Sll?

Sll.prototype.addFront = function(val) {
	var addNode = new Node(val);
	if (!this.head) {
		this.head = addNode;
	} else {
		addNode.next = this.head;
		this.head = addNode;
	}
}
