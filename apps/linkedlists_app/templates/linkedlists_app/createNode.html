Creat the Node class. A node class has two attributes: value and next. The "next" attribute will point to the next node in the Sll. When a new node is created, the value is the value passed to in and the next attribute is set to null. When creating a new node, create an instance of the node class var newNode = new Node;

function Node(val) {
	this.next = null;
	this.val = val;	
}
