[Delete From Back]

Input: N/A

Output: The Sll instance

Iterate through the given Sll and delete the last node. By looking back at the node constructor, we know that a newly created node will always have a next value of null. As new nodes are added, the last node in the chain points to the newly added node. Therefore, the last node in the Sll will have a next value of null. Using this logic, you are able to iterate through the Sll, locate the last node, and set the "next" attribute of the previous node to null. Once the association to the last node is removed, the node will be disassociated from the Sll and eventually removed from memory. 

Sll.prototype.delBack = function() {
        if (!this.head) {
            return false;
        } else if (!this.head.next) {
            this.head = null;
        } else {
            var runner = this.head;
            while (runner.next.next) {
                runner = runner.next;
            }
            runner.next = null;
        }
        return this;
    }
