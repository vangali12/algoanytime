[Add Node]

Input: Value of new node

Output: The Sll instance

Conceptually, the Sll data structure does not allow for quick retrieval of a specific node. Nodes are not indexed like the array data structure. Therefore, in order to add a new node, you must traverse through each of the elements in the Sll until reaching the end of the Sll, where the new node will be placed.

Finally, the return value should be the instance of the Sll to allow for chaining of node additions.

Edge Cases:

What do you do when you are given an empty Sll?

Sll.prototype.add = function(val) {
	var addNode = new Node(val);
	if (!this.head) {
		this.head = addNode;
	} else {
		var runner = this.head;
		while (runner.next) {
			runner = runner.next;
		}
		runner.next = addNode;
	}
	return this;
}
