[Add Node Values]

Input: A second instance of an Sll

Output: Integer - Sum of all Sll values

Combine the values that make up each of the Slls and create a number. For example, an Sll with the values 2 -> 7 -> 3 -> 1 become the number 2,731. AN Sll with the values 44 -> 13 -> 9 become 44,139. Given two instances of Slls, add the numbers up and return the total value of those Slls. Using the two examples given, the answer would be 46,870

Sll.prototype.addUp = function(second) {
	var fLen = this.length();
	var sLen = this.length();
	var fNum = 0;
	var sNum = 0;
	var newNum = 0;
	var runner = this.head;
	for (var i = fLen-1; i>=0; i--) {
		fNum += (runner.val)*Math.pow(10, i);
		runner = runner.next;
	}
	runner = second.head;
	for (var j = sLen-1; j>=0; i--) {
		sNum += (runner.val)*Math.pow(10, i);
		runner = runner.next;
	}
	newNum = fNum + sNum;
	var newList = new Sll();
	while (newNum>0) {
		newList.addFront(newNum % 10);
		newNum = Math.floor(newNum/10);
	}
	return newNum;
}
