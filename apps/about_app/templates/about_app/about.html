What to expect.

All of the algorithms on this site are coded in JavaScript. JavaScript just happens to be the language I was taught algorithms in. My dream is to hopefully include other languages (i.e. Java & C++) but for now, JavaScript it is.

On the far left column, you'll find the name of the algorithm, the expected input and output values, and a short description of the algorithm, it's purpose, it's capabilites and some common edge cases. Some algorithms will also contain an image or gif to assist with the understanding of the problem.

The middle column holds the code for that problem. Variable and function names are arbitrary and can be changed to whatever you please.

So take a gander, and use this site the way you'd like. I recommend you try coding out the problem before looking at the solution though. #StrengthThroughStruggle
