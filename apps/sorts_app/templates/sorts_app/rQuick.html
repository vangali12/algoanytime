[Recursive Quick Sort]

Input: Unsorted Array

Output: Sorted Array

Divide by choosing any element in the subarray array. Call this element the pivot. Rearrange the elements in arrayso that all other elements in array that are less than or equal to the pivot are to its left and all elements in array greater than the pivot are to the pivot's right. Recursively sort the newly created subarrays using the same method.

function rQuickSort(arr) {
    if(arr.length <= 1) return arr;

    let pivot = arr[0]; 
    let arr1 = [];
    let arr2 = [];

    for(let i = 1; i < arr.length; i++) {
        if(arr[i] < pivot) {
            arr1.push(arr[i]); 
        } else {
            arr2.push(arr[i]); 
        }
    }
    return(rQuickSort(arr1).concat(pivot).concat(rQuickSort(arr2)));
}
